* gptel

** Installation

Using straight to get the latest from GitHub.
#+begin_src emacs-lisp
    (use-package gptel
      :straight (:host github :repo "karthink/gptel"))

    (use-package transient
      :straight (:host github :repo "magit/transient"))
#+end_src
